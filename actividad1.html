Instalación del navegadores

Chromium

Para la instalación del navegador necesitamos abrir la terminal luego ingresamos el comando sudo apt install chromium Ejecutamos en comando y luego confirmamos la instalación con la letra "S" y comienza la instalación

Un tip, al escribir el comando sudo apt install chromium -y

Brave

NOTA: Brave no esta en los repositorios oficiales de debian, sigue los pasos para su instalación

Necesitas instalar sudo apt install curl -y si no lo tenemos, porque sirve para descargar archivos desde internet directamente en la terminal. Luego usas curl para bajar la clave oficial del programa, que Debian necesita para comprobar que es segura. Después añades el repositorio del programa para que Debian sepa dónde buscar sus paquetes. Finalmente ejecutas apt update y apt install para que el sistema descargue e instale el software desde ese repositorio confiable

sudo curl -fsSLo /usr/share/keyrings/brave-browser-archive-keyring.gpg https://brave-browser-apt-release.s3.brave.com/brave-browser-archive-keyring.gpg

Este comando descarga la clave de seguridad que firma los paquetes del programa. La clave se guarda en una carpeta especial donde Debian busca claves confiables. Sin esa clave, Debian no permite instalar nada del repositorio. Sirve para garantizar que los archivos no estén modificados ni falsificados.

echo "deb [signed-by=/usr/share/keyrings/brave-browser-archive-keyring.gpg] https://brave-browser-apt-release.s3.brave.com/ stable main" | sudo tee /etc/apt/sources.list.d/brave.list

Este comando crea un archivo nuevo donde Debian guarda direcciones de repositorios externos. Allí le dices dónde buscar las versiones oficiales del programa que quieres instalar. También le indicas qué clave debe usar para verificar esos paquetes. Sin agregar este repositorio, Debian no podría encontrar ni instalar ese software

sudo apt update

Este comando hace que Debian lea los repositorios que tiene agregados. Descarga la lista actualizada de programas disponibles desde cada fuente. Así el sistema sabe qué paquetes existen, qué versiones hay y desde dónde instalarlos. Es obligatorio después de agregar o modificar repositorios

sudo apt install brave-browser -y

Este comando instala el programa ahora que Debian ya sabe dónde encontrarlo. Descarga los archivos desde el repositorio oficial y los instala automáticamente. La opción -y solo evita que el sistema pregunte confirmaciones. Así obtienes la aplicación lista para usar sin pasos adicionales.
